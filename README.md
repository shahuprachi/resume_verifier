This patent proposes an intelligent system that verifies whether the skills mentioned in a candidate’s resume are genuine by analyzing behavioral response patterns during automated micro-assessments. The system extracts skills from uploaded resumes and dynamically generates short skill-based verification questions. It evaluates the candidate’s accuracy, response speed, and completion behavior to calculate an authenticity confidence score. This score helps recruiters identify exaggerated or incorrect skill claims before interviews. The system improves recruitment efficiency and ensures better hiring quality through automated behavioral skill verification.
